# 轨道方程

1. 轨道方程
\[
    r = \frac{h^2}{\mu}\frac{1}{1+e\cos{\theta}}    
\]
2. 速度方程
\[
    \begin{aligned}
    &v_r = \frac{\mu}{h}e\sin{\theta}   \\
    &v_\perp = \frac{\mu}{h}({1+e\cos{\theta}})
    \end{aligned} 
\]
3. 飞行路径角
\[
    \tan\gamma=\frac{v_r}{v_\perp}=\frac{e\sin{\theta}}{1+e\cos{\theta}}    
\]
4. 活力公式——能量守恒
    1. 单位质量的相对动能：$\frac{v^2}{2}$
    2. $m_2$在$m_1$引力场作用下的每单位质量所具有的势能：$-\frac{\mu}{r}$
- 比机械能
\[
    \varepsilon=\frac{v^2}{2}-\frac{\mu}{r}=-\frac{1}{2}\frac{\mu^2}{h^2}(1-e^2)
\]
5. 轨道特性公式
- 圆轨道（$e=0$）
    1. 速度
    \[v_圆=\sqrt{\frac{\mu}{r}}\]
    2. 周期
    \[T_圆=\frac{2\pi}{\sqrt{\mu}}r^{\frac{3}{2}}\]
    3. 比机械能
    \[\varepsilon=-\frac{1}{2}\frac{\mu^2}{h^2}=-\frac{\mu}{2r}\]
- 椭圆轨道（$0<e<1$）
    1. 偏心率
    \[e=\frac{r_a-r_p}{r_a+r_p}\]
    2. 半长轴
    \[2a=r_a+r_p\]
    3. 近地点/远地点
    \[r_a=a(1+e)，r_p=a(1-e)\]
    4. 比机械能
    \[\varepsilon=-\frac{1}{2}\frac{\mu^2}{h^2}{(1-e^2)}=-\frac{\mu}{2a}\]
    5. 周期
    \[T=\frac{2\pi}{\sqrt{\mu}}a^{\frac{3}{2}}\]
- 抛物线轨道（$e=1$）又称逃逸轨道
    1. 逃逸速度
    \[v_{逃逸}=\sqrt{\frac{2\mu}{r}}\]
    2. 比机械能
    \[\varepsilon=-\frac{1}{2}\frac{\mu^2}{h^2}{(1-e^2)}=0\]
    3. 飞行路径角
    \[\tan\gamma=\frac{v_r}{v_\perp}=\frac{\sin{\theta}}{1+\cos{\theta}}=\tan{\frac{\theta}{2}}    \]
- 双曲线轨道（e>1）真正能摆脱引力影响
    1. 近地点/远地点
    \[r_a=-a(e+1)，r_p=a(e-1)\]
    2. 半长轴
    \[a=\frac{|r_a|-r_p}{2}=\frac{h^2}{\mu}\frac{1}{e^2-1}\]
    3. 速度
    \[v^2=v_{\infty}^2+v^2_{逃逸}，其中v_{逃逸}=\sqrt{\frac{2\mu}{r}}\]
    4. 剩余速度
    \[v_{\infty}=\sqrt{\frac{\mu}{a}}=\frac{\mu}{h}e\sin{\theta_{\infty}}=\frac{\mu}{h}\sqrt{e^2-1}\]
    5. 特征能量
    \[C_3=v_{\infty}^2\]
    6. 准径（之后又叫瞄准半径）
    \[\Delta = a\sqrt{e^2-1}\]
    7. $\beta$：渐近线与准线夹的锐角
    \[\beta=180^\circ-\theta_{\infty}=\arccos{\frac{1}{e}}\]
    8. 转向角
    \[\delta=180^\circ-2\beta=2arcsin{\frac{1}{e}}\]
